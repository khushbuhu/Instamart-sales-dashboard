# Swiggy Instamart Sales Analysis Dashboard | Power BI

## 📌 Project Overview
This Power BI dashboard analyzes Swiggy Instamart sales performance, customer ratings, order trends, and category-wise insights. The dashboard helps in understanding business performance across different city tiers and product categories.

## 📊 Key Insights
- Analyzed 8.5K+ orders and 1.20M+ sales data
- Identified top-performing product categories and city tiers
- Tracked sales trends, average sales, and customer ratings
- Highlighted category-wise and outlet-wise business performance

## ⚙️ Tools & Technologies Used
- Power BI
- Power Query
- DAX
- Data Modeling
- Field Parameters
- Interactive Slicers

## ✨ Key Features
- Dynamic KPI switching using Field Parameters
- Interactive filters and slicers
- Sales and rating trend analysis
- Tier-wise and category-wise performance tracking
- User-friendly dashboard UI design
